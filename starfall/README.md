Node.js 기반 Youtube 음악 및 썸네일 다운로드 -> Starfall에서 스트리밍

# 기능

1.  `!song {유튜브 URL}` : 큐에 음악 추가
2.  `!volume {볼륨}` : 볼륨 수정
3.  `radius {범위}` : 범위 수정
4.  `!time {시간}` : 재생 시간 변경
5.  `!play` : 재생
6.  `!pause` : 일시정지
7.  `!stop` : 정지
<!-- 8.  `!shuffle` : 셔플 설정 (on/off) -->
8.  `!lock` : 조작 잠금
9.  `!loop` : 반복 설정 (반복 없음/전체 반복/단일 반복)
10. `!next` : 다음 곡
11. `!prev` : 이전 곡으로 재생 (기록이 있을 시)

---

음악 큐 필요
다음 곡으로 전환 시 기존 곡을 재생 스택에 push 필요 (이전 곡 재생용)
서버 측에서 요청 처리가 가능한지 확인 필요
SERVER -> CLIENT 명령 하달 방식

## SERVER

- 명령어 감지 및 처리
- 음악 큐 저장
- 재생 기록 스택 저장
- 클라이언트로 노래 UUID 전송
- 클라이언트로 노래 제어 명령 전송
- 클라이언트(owner)로부터 수신한 시간 정보 기반 다음 곡 요청 명령 전송

## CLIENT

- 서버로부터 수신한 노래 UUID로 bass 재생
- 서버로부터 수신한 노래 제어 명령으로 bass 속성 변경
- 서버로 시간 정보 반복 송신 (owner 클라이언트만, 4Hz)
