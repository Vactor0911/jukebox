Node.js 기반 Youtube 음악 및 썸네일 다운로드 -> Starfall에서 스트리밍

# 기능

1. 큐에 음악 추가 (다운로드)
   1. Youtube 영상 링크 전송
   2. 링크 내 v 파라미터 값 추출
   3. MariaDB에서 기존 데이터 존재 여부 확인 (있다면 기존 파일 경로 반환 후 완료 응답)
   4. 영상 검증 (길이 6분 미만, 용량 10MB 미만)
   5. 음성(mp3) 및 썸네일(jpg) 다운로드 후 /mnt/music 에 저장, 파일명은 UUIDv4로 지정
   6. Youtube 영상 링크 내 v 파라미터 값, 저장위치 + 파일명을 MariaDB에 INSERT
   7. 파일 경로 + 완료 메시지 응답
2. 재생, 일시정지, 정지, 재생 시점 변경
   1. bass 관련 기능 활용 직접 구현
3. 1주일 이상 재생되지 않은 곡 삭제
